Trainee Projects

This website was built from scratch using HTML, CSS, and JavaScript. The goal was to showcase my projects, skills, and contact information in a clean, responsive layout. I used Flexbox and media queries to ensure it looked great on all devices. Through this project, I learned about page structure, design consistency, and how to make a website user-friendly.

I was tasked with creating a furniture and interiors website. I used Bootstrap to speed up development and made use of pre-built components like carousels and modals. I also added animations with CSS and made sure the site passed basic accessibility checks. This project helped me understand real-world design requirements and taught me how to balance style with performance.

As a trainee at Hot Beans Web, I built a tech job recruitment site using HTML, CSS, JavaScript, and PHP. It featured job listings, a filter system, and an application form that saved data to a database. The site was fully responsive and designed to connect developers with hiring companies. This project improved my full-stack skills and understanding of user-focused design.
